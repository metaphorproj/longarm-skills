# longarm

longarm is an Android automation and remote-control app. It lets you control a real phone or tablet from another device on the same network by sending taps, long presses, swipes, screenshots, app launches, intents, and repeatable batch tasks.

It is useful for developers, testers, and AI agents that need scriptable control of an Android device without root. longarm also includes a built-in MCP server so compatible AI tools can inspect the screen and call device-control actions directly.
